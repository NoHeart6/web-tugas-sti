Saya adalah mahasiswa baru Unisnu yang mengambil jurusan Teknik Informatika. Saya tertarik dengan bidang web development karena saya ingin membuat aplikasi web yang bermanfaat dan menarik. Saya memiliki pengalaman dalam menggunakan HTML, CSS, JavaScript, dan PHP untuk membuat web statis dan dinamis. Saya juga pernah mengikuti kursus online tentang web design dan web security. Salah satu proyek web yang saya buat adalah sebuah web toko online yang menjual produk-produk lokal. Web ini memiliki fitur seperti keranjang belanja, pembayaran online, dan sistem rating. Saya berharap dengan portofolio ini, saya dapat menunjukkan kemampuan dan potensi saya sebagai web developer.
